# SpotPitch

Real-time pitch and speed control for Spotify on Mac.

## Requirements
- macOS (arm64)
- Spotify installed in /Applications

## How to use
1. Double click `SpotPitch.command`
2. Spotify will open automatically
3. Play any song
4. Use the sliders to control pitch and speed

## Presets
- **Normal** — original pitch and speed
- **Slowed** — slowed down with lower pitch
- **Nightcore** — sped up with higher pitch  
- **Daycore** — slower with lower pitch

## Notes
- Close the SpotPitch window to exit
- Re-run SpotPitch.command each time you want to use it
- Spotify updates may require re-running SpotPitch
